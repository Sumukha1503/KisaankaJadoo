# 🌾 Kisaan Ka Jadoo

**Kisaan Ka Jadoo** is an AI-powered smart farming platform designed to help farmers make informed decisions using technology. The application provides crop recommendations, disease detection, weather insights, and farming guidance in a simple, user-friendly interface.

## 🚀 Features

* 🌱 **Crop Recommendation**

  * Suggests suitable crops based on soil type, location, season, and environmental conditions.

* 📷 **Plant Disease Detection**

  * Detects plant diseases from uploaded leaf images using AI and provides treatment suggestions.

* 🌦️ **Weather Information**

  * Displays current weather conditions and forecasts to assist in farming decisions.

* 💧 **Fertilizer & Irrigation Guidance**

  * Recommends fertilizers and irrigation schedules based on crop requirements.

* 🤖 **AI Farming Assistant**

  * Answers farmers' questions in natural language and provides practical farming advice.

* 🌍 **Multi-language Support**

  * Designed to support regional languages for better accessibility.

* 📱 **Responsive Design**

  * Works seamlessly on desktops, tablets, and mobile devices.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/kisaan-ka-jadoo.git
```

Navigate to the project folder:

```bash
cd kisaan-ka-jadoo
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add your environment variables:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_WEATHER_API_KEY=your_weather_api_key
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
